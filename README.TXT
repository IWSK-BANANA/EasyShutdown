1. What is This !?　～ これってなんやねん

　これは パソコンのシャットダウン操作をもっとわかりやすくして～な～ と思った結果夕方テンションでチャチャっと、C++で作りました。
　WindowsのshutdownコマンドをGUi操作でやりたかっただけなので特別な機能はないです。

---------------------------------------------------------------------------------------------------
2. 必要スペック

　必要最低スペック：
    OS: Windows XP以降（多分）（shutdownコマンドが使えるWindowsだったらおｋ）
    ほか: OSの最低スペックで使える（と思う）
    もしかしたら、VC++のランタイムがいるやもしれん。そん時は、同封のVC_redistなんとか.exeを実行してください。VC++の再頒布可能パッケージの中身がインストールされます。

---------------------------------------------------------------------------------------------------
3. 使い方（意味浅）　～ べっ、別に、特別な意味なんてないんだからねっ！

　実行するとウィンドウが出ます。
　ボタン、チェックボックス、メニューバーがあるんでそれを見てなんとなく あーはいはい、あれね。 と察してください。
